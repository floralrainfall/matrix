# MATRIX 3D

Matrix is a 3D engine. It is the successor to SGLThing, and its name
used to be SGLThing++.

Matrix supports (at the present)

* OpenGL 3.0
* Vulkan (in development)
* Networking (via ENet)
* BSP (Quake 3 format)
* Sound with irrKlang
* HTTP support using MatrixWorld/CURL
* A rudimentary material system
